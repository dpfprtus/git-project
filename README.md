
# Git 자주 사용하는 명령어 정리
자주 사용하는 기본적인 명령어들을 정리했습니다.

## 저장소 생성
'''bash
git init #저장소 생성
'''

##커밋하기
'''bash
git add README.md #커밋할 파일 추가
git status
git commit -m "~"
git log --decorate=full --oneline --graph # 커밋로그 확인
'''

##브랜치 생성하기
'''bash
git checkout -b readme #readme 라는 이름의 브랜치 생성
'''
